# Smart Study Scheduler 🎓

A comprehensive full-stack study planning application that helps students organize their study time effectively using smart scheduling algorithms.

## 🚀 Features

- **Smart Scheduling**: AI-powered study schedule generation based on task priority and deadlines
- **Task Management**: Create, track, and manage study tasks with priority levels
- **Progress Tracking**: Visual analytics and productivity insights
- **Calendar Integration**: Interactive calendar view of your study schedule
- **Reminders & Notifications**: Email and real-time notifications for upcoming tasks
- **Achievement System**: Gamified learning with streaks and achievements
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile

## 🛠️ Tech Stack

### Backend
- Node.js & Express.js
- MongoDB with Mongoose
- JWT Authentication
- Socket.io for real-time updates
- Redis for caching
- Nodemailer for emails
- Puppeteer for PDF generation

### Frontend
- React 18 with Vite
- Tailwind CSS for styling
- Recharts for analytics
- React Big Calendar
- Framer Motion for animations
- React Query for data fetching
- Socket.io-client
